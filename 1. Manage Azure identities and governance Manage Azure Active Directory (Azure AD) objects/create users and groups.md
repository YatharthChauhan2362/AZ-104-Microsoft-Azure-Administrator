To create users and groups in Azure Active Directory (Azure AD), you can follow these steps:

1.  Sign in to the Azure portal (portal.azure.com) with your Azure AD administrator account.
<img width="960" alt="image" src="https://github.com/YatharthChauhan2362/AZ-104-Microsoft-Azure-Administrator/assets/75237577/8a1181a6-f543-405b-a606-aa9d6641941a">

2.  In the left-hand menu, select "Azure Active Directory."
<img width="960" alt="image" src="https://github.com/YatharthChauhan2362/AZ-104-Microsoft-Azure-Administrator/assets/75237577/b55bc0f7-9e84-4734-acf9-8526d52c41e1">

3.  In the Azure Active Directory blade, select "Users" to create users or "Groups" to create groups.
<img width="960" alt="image" src="https://github.com/YatharthChauhan2362/AZ-104-Microsoft-Azure-Administrator/assets/75237577/62981b33-857f-4e18-a86f-2a557d79455a">



Creating a User:

1.  In the "Users" blade, select "New user."
<img width="960" alt="image" src="https://github.com/YatharthChauhan2362/AZ-104-Microsoft-Azure-Administrator/assets/75237577/6b2438ef-7292-464a-bc97-4e201ce712fb">

2.  Provide the required user details such as user name, name, and domain (if applicable) and Set the user's sign-in credentials, such as username and password.
<img width="960" alt="image" src="https://github.com/YatharthChauhan2362/AZ-104-Microsoft-Azure-Administrator/assets/75237577/2fb2096e-89d0-43bf-83de-314f58a18c19">

3.  Configure additional settings as needed, such as assigning licenses or roles.
<img width="960" alt="image" src="https://github.com/YatharthChauhan2362/AZ-104-Microsoft-Azure-Administrator/assets/75237577/132309c1-42c8-4d66-9542-d7383ccfbeea">

4.  Click "Create" to create the user.
<img width="339" alt="image" src="https://github.com/YatharthChauhan2362/AZ-104-Microsoft-Azure-Administrator/assets/75237577/eaa0d4c9-8503-48a6-8bc7-2071a69995f0">


Creating a Group:

1.  In the "Groups" blade, select "New group."
<img width="957" alt="image" src="https://github.com/YatharthChauhan2362/AZ-104-Microsoft-Azure-Administrator/assets/75237577/c7a603fb-10c3-4262-9a53-98fae18c1ead">

2.  Specify the group name, group type (security or Microsoft 365), and any other required information.
<img width="960" alt="image" src="https://github.com/YatharthChauhan2362/AZ-104-Microsoft-Azure-Administrator/assets/75237577/8bdf2fd6-747a-48da-a1b2-35edff3f4e3f">

3.  Add members to the group by selecting users from the available list or searching for specific users.
<img width="960" alt="image" src="https://github.com/YatharthChauhan2362/AZ-104-Microsoft-Azure-Administrator/assets/75237577/91fde112-cd96-4df8-a358-edc262dcabca">

4.  Set group settings like membership type and group owners.
<img width="960" alt="image" src="https://github.com/YatharthChauhan2362/AZ-104-Microsoft-Azure-Administrator/assets/75237577/9732fea5-da1d-4ef5-b932-d45aa50552d9">

5.  Click "Create" to create the group.
<img width="345" alt="image" src="https://github.com/YatharthChauhan2362/AZ-104-Microsoft-Azure-Administrator/assets/75237577/dbd10f9b-fd82-4402-a219-0172b5a709f5">


Remember to provide appropriate permissions and access controls to the users and groups as necessary.

These steps should help you create users and groups in Azure AD.
